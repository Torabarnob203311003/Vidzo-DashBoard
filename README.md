# Vidzo Dashboard

## Development

- Install deps: `npm i`
- Run dev server: `npm run dev`

## Responsive behavior

- **Mobile sidebar:** Use the hamburger menu in the top header to open/close the sidebar.
- **Dashboard pages:** Page padding and grid layouts adapt from mobile → desktop (`sm`/`lg` breakpoints).
- **Messages:** On small screens, the conversation list is shown first; selecting a conversation switches to chat with a back button.
